# ShowcaseGG
 
## Description
ShowcaseGG is a web app that turns your Steam profile into a clean, collectible-style profile card. Enter your Steam vanity URL or SteamID to generate a card displaying your most played games, total hours, profile picture, and a QR code linking to your Steam profile. Cards can then be customized and saved to a global gallery for others to view.
 
## Target Browsers
This app is designed and tested for **Google Chrome** on desktop. Other browsers may work but they are not tested.
